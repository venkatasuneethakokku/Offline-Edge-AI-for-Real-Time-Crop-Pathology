# Offline Edge AI for Real-Time Crop Pathology

An intelligent, fully offline crop disease detection system powered by Edge AI and TensorFlow Lite. This application enables farmers and agricultural professionals to identify crop diseases instantly using on-device image analysis—without requiring an internet connection.

## Key Features

* Real-time crop disease detection using TensorFlow Lite
* Fully offline inference for remote and low-connectivity environments
* FastAPI backend for efficient model serving
* Responsive and user-friendly frontend interface
* Detailed disease insights including symptoms, treatment, and prevention
* Lightweight, scalable, and optimized for edge deployment

## Tech Stack

* Python
* FastAPI
* TensorFlow Lite
* NumPy
* Pillow
* HTML, CSS, JavaScript

## Use Case

Designed for precision agriculture, this solution helps farmers make timely and informed decisions by providing rapid, accurate, and accessible plant disease diagnosis directly on edge devices.

## Impact

* Reduces dependency on internet connectivity
* Enables low-latency on-device predictions
* Supports sustainable and smart farming practices
* Improves early disease detection and crop health management
